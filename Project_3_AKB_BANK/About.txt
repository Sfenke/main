
Задача 1. Есть массив чисел от 1 до 100000000000. Массив отсортирован(1,2,3 ....99999999999,100000000000 ), но в нем  одна цифра задвоилась(Например, [1,2,3,4,4,6,7 ...]). Надо написать метод который найдет отсутствующее число за минимальное время.

Задача 2. Есть файл с текстом, надо написать метод который выведет каждое третье слово из текста.


Задания сделанны на языке Java в NetBeans IDE 8.2.

1. Задание. Проиллюстрирован алгоритм нахождения пропущенного числа.

Для тестов использован массив меньшего размера 30 000 000.

2. Задание. Использован текст text.txt. Слова являющиеся 3 по счету подписанны.
